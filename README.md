# PoC - Proof of concept

Install node v22 using [nvm](https://github.com/nvm-sh/nvm)

```shell
nvm install 22
```

Use node 22

```shell
nvm use 22
```

Install the dependencies

```shell
npm i
```

Run the app locally

```shell
npm run dev
```
